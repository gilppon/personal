# SNS 성장 전략

> 1인 기업의 SNS 채널 성장을 위한 실전 전략

## 📱 채널별 전략

### YouTube
- **목표**: 검색 기반 상시 유입 + 브랜드 권위
- **핵심 지표**: 시청 시간 > 조회수 > 구독자
- **콘텐츠**: 튜토리얼, 비하인드, 리뷰, 브이로그
- **빈도**: 주 1~2회 (일관성이 핵심)
- **SEO**: 제목 키워드, 설명 챕터, 자막 파일

### Instagram
- **목표**: 브랜드 인지도 + 커뮤니티 구축
- **핵심 지표**: 저장 > 공유 > 댓글 > 좋아요
- **콘텐츠**: 릴스(도달), 캐러셀(저장), 스토리(소통)
- **빈도**: 릴스 주 3~5회, 스토리 매일
- **해시태그**: 9~15개 (대형3 + 중형3 + 소형3)

### 블로그 (Naver/Google)
- **목표**: 롱테일 검색 유입 + 자산화
- **핵심 지표**: 검색 노출, 클릭률, 체류 시간
- **콘텐츠**: How-to, 비교, 리스트, 경험담
- **빈도**: 주 2~3회
- **SEO**: H2 구조화, 이미지 alt 태그, 내부 링크

## 🔄 콘텐츠 재활용 공식 (1→5)
```
블로그 글 (원본)
  ├→ YouTube 영상 (스크립트 기반)
  ├→ Instagram 캐러셀 (핵심 5장)
  ├→ Instagram 릴스 (30초 요약)
  └→ 트위터/스레드 (핵심 포인트)
```

## 📈 성장 단계별 전략

### Stage 1: 0~1K 팔로워
- 니치에 집중, 넓게 퍼지지 않기
- 매일 관련 계정에 진성 댓글 10개+
- 콘텐츠 품질 > 빈도

### Stage 2: 1K~10K 팔로워
- 콜라보/게스트 콘텐츠 시작
- 이메일 리스트 구축 시작
- 광고/프로모션 소규모 테스트

### Stage 3: 10K+ 팔로워
- 커뮤니티 운영 (디스코드, 오픈카톡)
- 유료 콘텐츠/서비스 런칭
- 파트너십/스폰서십 협상
