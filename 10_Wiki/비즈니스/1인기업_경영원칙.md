# 1인 기업 경영 원칙

> 넥스트하루의 핵심 경영 철학과 운영 원칙

## 🧭 5대 경영 원칙

### 1. 가치 우선 (Value First)
- 돈을 쫓지 말고, 문제를 해결하라
- "이 서비스가 없어지면 누가 슬퍼할까?"를 매주 자문
- 가치가 증명되면 돈은 자연스럽게 따라온다

### 2. 니치 전문화 (Niche Down)
- 모든 사람을 위한 서비스는 아무도 위한 서비스가 아니다
- "1만 명이 좋아하는 것"보다 "100명이 사랑하는 것"
- 니치를 장악한 후 확장 (Amazon도 책에서 시작)

### 3. 린(Lean) 실행
- MVP → 피드백 → 개선 사이클을 최대한 빠르게
- 완벽한 제품보다 빠른 출시가 우선
- 고객 10명의 피드백 > 혼자 3개월 개발

### 4. 자동화 최우선
- 반복 작업은 즉시 자동화 대상
- 수동 작업 3회 반복 → 자동화 검토
- AI + 자동화로 1인이 10인 효과

### 5. 지속 가능성
- 번아웃 방지: 주 40시간 이내 목표
- 수익의 20%는 재투자 (도구, 학습, 마케팅)
- 긴급하지 않지만 중요한 일에 매주 4시간 배정

## 📊 주간 경영 리뷰 체크리스트
```
매주 금요일 오후 5시
- [ ] 이번 주 매출/지출 확인
- [ ] 핵심 KPI 변동 확인 (트래픽, 전환율, 이탈률)
- [ ] 고객 피드백 정리 (긍정/부정 TOP 3)
- [ ] 다음 주 우선순위 3개 설정
- [ ] decisions.md에 주요 결정 기록
```

## ⏰ 시간 관리 매트릭스 (아이젠하워)
| | 긴급 | 긴급하지 않음 |
|--|------|-------------|
| **중요** | 즉시 실행 (장애, 데드라인) | 계획 후 실행 (전략, 학습) |
| **중요하지 않음** | 위임/자동화 (이메일, 알림) | 제거 (SNS 무한스크롤) |

## 💡 1인 기업 생존 규칙
1. **현금흐름이 왕**: 매출 없이 6개월 버틸 런웨이 확보
2. **고정비 최소화**: 구독 서비스 분기별 점검, 미사용 해지
3. **학습 = 투자**: 매주 최소 5시간은 스킬 업그레이드
4. **네트워크 = 자산**: 같은 분야 1인 기업 커뮤니티 참여
5. **건강 = 인프라**: 운동, 수면, 식사 — 본인이 서버이므로 다운 금지
