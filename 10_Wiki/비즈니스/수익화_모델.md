# 수익화 모델

> 1인 기업의 수익 구조 설계 가이드

## 💰 수익 모델 비교

### 1. SaaS 구독 모델
- **장점**: 예측 가능한 MRR(월간 반복 매출), 높은 LTV
- **단점**: 이탈률 관리 필요, 지속적 기능 업데이트
- **적합**: 도구형 웹앱, 대시보드, 자동화 서비스
- **가격 구조**: Free / Pro($9.9) / Team($29.9)

### 2. 디지털 제품 판매
- **장점**: 제작 후 추가 비용 없음, 높은 마진
- **단점**: 지속적 마케팅 필요, 환불 관리
- **적합**: 템플릿, 전자책, 강의, 디자인 에셋
- **플랫폼**: Gumroad, 탈잉, 클래스101

### 3. 프리미엄 (Freemium)
- **장점**: 낮은 진입 장벽으로 사용자 확보
- **단점**: 무료→유료 전환율 2~5%로 낮음
- **핵심**: 무료로 가치를 증명, 유료로 10배 가치 제공
- **제한 방식**: 기능 제한 / 용량 제한 / 기간 제한

### 4. 광고 수익
- **장점**: 사용자 부담 없음
- **단점**: 대규모 트래픽 필요 (최소 10만 PV/월)
- **적합**: 블로그, 유튜브, 무료 도구
- **CPM 기준**: 한국 ₩1,000~5,000 / 미국 $3~10

### 5. 컨설팅/서비스
- **장점**: 즉시 수익, 고단가
- **단점**: 시간 = 돈 (스케일 한계)
- **전략**: 제품화 가능한 영역으로 점진적 전환

## 📊 가격 결정 체크리스트
- [ ] 타깃 고객의 지불 의향 조사 (설문/인터뷰)
- [ ] 경쟁사 가격 3곳 이상 조사
- [ ] 3-Tier 구조 설계 (Basic/Pro/Enterprise)
- [ ] 연간 결제 할인 적용 (20~30%)
- [ ] 무료 체험 기간 설정 (7일 또는 14일)

## 🎯 핵심 공식
```
LTV (생애가치) = ARPU × 평균 구독 기간
CAC (획득 비용) = 마케팅 비용 ÷ 신규 고객 수
목표: LTV > 3 × CAC
```
