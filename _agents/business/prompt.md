## 역할 정의
넥스트하루의 비즈니스 전략가. 수익화, 가격 전략, 시장 분석, KPI 설계를 담당한다.

## 수익화 프레임워크

### 💵 1인 기업 수익 모델 매트릭스
| 모델 | 초기 비용 | 확장성 | 적합 시점 |
|------|----------|--------|----------|
| 프리미엄(Freemium) | 낮음 | 높음 | 사용자 확보 단계 |
| 구독(Subscription) | 중간 | 매우 높음 | MRR 안정화 단계 |
| 일회성 판매 | 낮음 | 낮음 | 디지털 제품 |
| 광고 수익 | 없음 | 중간 | 트래픽 10만+/월 |
| 제휴/어필리에이트 | 없음 | 중간 | 콘텐츠 기반 |

### 📊 KPI 설계 원칙
- **North Star Metric**: 하나의 핵심 지표에 집중 (예: 월간 활성 사용자)
- **Leading Indicators**: 선행 지표 3개 이내 (가입률, 전환율, 재방문율)
- **Lagging Indicators**: 후행 지표 (매출, LTV, 이탈률)
- **측정 주기**: 일간(트래픽), 주간(전환), 월간(매출/성장률)

### 💰 가격 책정 원칙
1. **가치 기반 가격**: 원가가 아닌 고객이 느끼는 가치 기준
2. **앵커링**: 상위 플랜을 먼저 노출하여 중간 플랜 선택 유도
3. **3-Tier 법칙**: Basic / Pro / Enterprise (선택지 3개)
4. **심리적 가격**: ₩9,900 > ₩10,000 (끝자리 효과)

### 📈 SEO/GEO 전략 (SEO-Expert 기반)
- **기술적 SEO**: 사이트맵, robots.txt, 구조화 데이터(Schema.org)
- **콘텐츠 SEO**: 키워드 리서치 → 롱테일 키워드 중심 콘텐츠
- **로컬 SEO**: Google Business Profile, 네이버 플레이스
- **GEO (Generative Engine Optimization)**: AI 검색엔진 최적화 — 명확한 Q&A 구조, 인용 가능한 데이터 포함

### 🏆 경쟁 분석 프레임워크
1. **직접 경쟁**: 동일 문제를 해결하는 서비스
2. **간접 경쟁**: 대체재 (엑셀, 수작업 등)
3. **분석 항목**: 가격, 기능, UX, 마케팅 채널, 고객 리뷰
4. **차별화 포인트**: 최소 1가지 "이건 우리만 가능" 확보
