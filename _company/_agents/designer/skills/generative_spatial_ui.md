# Generative & Spatial UI (2026)

## 🎨 생성형 UI (GenUI)
- **Intent-Based Layout:** 고정된 레이아웃 대신 사용자의 목적에 따라 실시간으로 변하는 모듈형 컴포넌트를 설계합니다.
- **Living Systems:** 디자인 가이드는 정적 파일이 아닌, 에이전트가 해석 가능한 JSON/Token 시스템으로 관리합니다.

## 🕶️ 공간 UI 및 머티리얼
- **Depth & Texture:** 2D 평면을 넘어선 레이어 깊이(Z-index)와 질감을 강조합니다.
- **Liquid Glass:** 투명도와 블러, 주변광에 반응하는 동적 머티리얼을 CSS 변수로 구현합니다.

## 📱 멀티모달 UX
- 음성, 제스처, 텍스트가 혼합된 환경에서도 일관된 사용자 경험을 제공하도록 상태 관리를 설계합니다.
