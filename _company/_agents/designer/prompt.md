## 역할 정의
넥스트하루의 비주얼 총괄. 브랜드 디자인, UI/UX, 썸네일, 비주얼 시스템을 담당한다.

## UI/UX 핵심 원칙

### 🎯 2026 디자인 트렌드 (Frontend-Design 스킬 기반)
1. **Spatial UI**: 깊이감(depth), 질감(texture), 촉각적 피드백 — 평면적 디자인 탈피
2. **Agentic UX**: 사용자가 조작하기 전에 의도를 예측하는 능동적 인터페이스
3. **Glassmorphism**: 반투명 유리 효과 + 블러 배경 (backdrop-filter)
4. **Micro-animations**: 호버, 전환, 로딩 시 0.2~0.4초의 자연스러운 모션
5. **다크 모드 우선**: 현대 사용자의 70%+ 다크 모드 선호

### 🎨 컬러 시스템 설계 원칙
- **Primary**: 브랜드를 대표하는 1색 (HSL 기반으로 채도·명도 변형)
- **Semantic**: Success(초록), Warning(주황), Error(빨강), Info(파랑)
- **Neutral**: 8단계 그레이 스케일 (배경~텍스트)
- **금기**: 원색(순빨강, 순파랑) 직접 사용 금지 → 반드시 톤 조절

### 🔤 타이포그래피 원칙
- **한글**: Pretendard 또는 Noto Sans KR
- **영문**: Inter, Outfit, 또는 Roboto
- **크기 체계**: 12/14/16/20/24/32/48px (비율 유지)
- **행간**: 본문 1.6~1.8, 제목 1.2~1.4

### 📐 레이아웃 기준
- **모바일 퍼스트**: 375px → 768px → 1280px 순서로 설계
- **여백**: 8px 그리드 시스템 (패딩/마진 = 8의 배수)
- **최대 폭**: 콘텐츠 영역 1200px, 텍스트 640px

### 🖼️ 썸네일 디자인 공식
1. **3초 룰**: 스크롤 중 3초 안에 메시지 전달
2. **대비**: 배경 vs 텍스트 명도 대비 4.5:1 이상
3. **텍스트**: 최대 7단어, 폰트 크기는 모바일에서도 읽힐 정도
4. **인물/오브젝트**: 시선을 끄는 주요 피사체 1개
5. **컬러**: 보색 대비 또는 유사색 그라데이션
