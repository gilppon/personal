## 🛡️ 2026 Agentic Core Directive (Harness System)
1. **Design-Before-Code**: 코드를 작성하거나 제안하기 전에 반드시 디자인 의도(색상 팔레트, 서체, 공간감)를 먼저 브리핑한다.
2. **Spatial-First**: 모든 디자인 결정에서 '평면(Flat)'을 탈피하고 '공간감(Depth)'을 최우선으로 고려한다.
3. **Verification Gate**: UI 제안 후에는 반드시 대표님의 '시각적 검증' 단계를 거치며, 실패 시 Circuit Breaker 프로토콜에 따라 대안을 보고한다.

## 💎 2026 Spatial Design Standards
1. **Z-Axis Composition**: `backdrop-filter: blur()`, 다층 그림자(Multi-layered shadow)를 활용하여 요소가 물리적 공간에 부유하는 느낌을 구현한다.
2. **Tactile Texture**: 단색 배경을 지양하고, 노이즈 오버레이 및 메쉬 그라디언트로 촉각적 질감을 부여한다.
3. **HSL Color Engineering**: 랜덤 HEX 코드 사용을 금지하고, 수치적으로 설계된 HSL 기반 팔레트를 구축한다.
4. **Micro-Animation**: 모든 상호작용에 0.2~0.4s의 `cubic-bezier` 트랜지션을 적용하여 리듬감 있는 피드백을 제공한다.

## 🎯 UI/UX 핵심 원칙 (Legacy & Core)
- **모바일 퍼스트**: 375px → 768px → 1280px 설계 순서 준수.
- **8px 그리드**: 모든 여백과 크기는 8의 배수로 설계 (Padding/Margin = 8n).
- **타이포그래피**: 한글(Pretendard), 영문(Outfit/Inter) 사용. 시스템 폰트 직접 사용 금지.
- **금기**: 원색(순빨강, 순파랑) 직접 사용 금지 → 반드시 톤 조절된 Semantic 컬러 사용.
