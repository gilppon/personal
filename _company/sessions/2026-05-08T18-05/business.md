### 💡 최종 미션: A/B 테스트를 위한 지표-데이터 매트릭스 완성 및 API 사양 확정
(가설 검증 단계로 진입하기 위해, 이론적 계획을 코드로 변환하는 작업)

#### 🎯 1단계: A/B 테스트 가설 기반 KPI 정의 (💰 Business 주도)
단순히 '클릭률'을 측정할 것이 아니라, **어떤 메시지(A vs B)가 고객의 어떤 심리 상태(Pain Point 해결)를 건드려 어느 행동(CTA 클릭)으로 이어지는지**에 대한 인과관계를 설정해야 합니다.

| 테스트 변수 (Variable) | 가설 (Hypothesis) | 핵심 지표 (Primary KPI) | 측정 목표 (Target/Benchmark) | 기대 ROI 효과 |
| :--- | :--- | :--- | :--- | :--- |
| **메시지 톤앤매너** | A: 손실 강조(Pain Focus)가 B: 성과 제시(Outcome Focus)보다 더 즉각적 행동 유도에 강력할 것이다. | CTA 클릭률 (CTR) / 이탈 지점 전 회귀율 (Bounce Back Rate at Exit Point) | CTR 12% 증가, Bounce Back Rate 5% 이상 확보 | 초기 구매 결정 가속화 및 전환율 극대화 |
| **가격 옵션 구조** | B: 'Accelerator'와 같은 중간 번들(Bundle)을 핵심으로 제시하는 것이 'Starter/Scale'의 이분법적 대비보다 높은 평균 객단가(AOV)를 유도할 것이다. | Average Order Value (AOV) / Bundle 선택 비율 | AOV 20% 증가, Accelerator 선택률 > 45% | 고객 생애 가치(LTV) 증대 및 수익성 확보 |
| **CTA 문구** | "지금 시작하세요"보다 "**[Pain Point]를 방지하려면 지금 바로 시작해야 합니다.**"와 같이 손실 회피 심리를 자극하는 문구가 더 강력할 것이다. | Exit Rate (페이지 이탈률) / CTA 클릭 직전 체류 시간(Dwell Time) | Exit Rate 8% 감소, Dwell Time 15초 이상 유지 | 행동 촉발 타이밍 최적화 및 즉각적인 실행 유도 |

#### 💻 2단계: 데이터 수집 API 사양 확정 (Developer 협업 필수)
위의 KPI를 측정하기 위해 개발팀에게 필요한 데이터를 **어떤 시점(Event)**에, **어떤 속성값(Property)**을 담아 전송해야 하는지 명확한 스키마로 정의합니다.

**[필수 데이터 로그 항목 및 API 연동 요청]**

1.  **`user_behavior_logs` 테이블 확장/검증:**
    *   **요청 사항:** 기존의 단순 클릭 기록을 넘어, **'스크롤 깊이(Scroll Depth)'**와 **'체류 시간(Dwell Time)'** 데이터를 세션별로 수집하고 `user_behavior_logs`에 통합해야 합니다.
    *   **데이터 활용:** 사용자가 어느 지점(Pain Section인지, Solution Section인지)에서 이탈하는지 파악하여 메시징을 수정할 근거를 마련합니다.

2.  **A/B 테스트 이벤트 상세 로깅 (가장 중요):**
    *   **요청 사항:** `event_id=CTA_Test_[A|B|C]` 발생 시, 다음의 속성을 필수로 수집해야 합니다.
        *   `test_group`: A, B, C 중 어떤 그룹에 배정되었는지 (필수)
        *   `message_type`: 'Pain Focus' 또는 'Outcome Focus' 중 어떤 메시지인지 (가설 검증용)
        *   `price_bundle`: 사용자가 본/선택한 가격 구조 ('Starter', 'Accelerator', 'Scale')
        *   `impression_count`: 해당 CTA를 본 횟수

3.  **전환 퍼널 이탈 지점 로그 추가:**
    *   **요청 사항:** 랜딩 페이지의 주요 섹션(가격/특징 소개 등)을 마이크로 인터랙티브 존으로 나누고, 각 섹션을 벗어날 때 **'이탈 이벤트(Exit Event)'**를 트리거하여 어떤 콘텐츠가 사용자 이탈의 원인이 되었는지 추적해야 합니다.
